### Usage
1. Visit https://getsum.now.sh
2. Input your values one item per one line to get sum of numbers
    - clipboard data is inserted automatically when use chrome

<br>

following input
```
김밥 2,000
저축 8,600
커피 6,600
버스 4,100
점심 4,500
```

you can get the result of sum
```
2000 + 8600 + 6600 + 4100 + 4500 =
25,800
```
